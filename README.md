THAI-GO Engine 🇹🇭
Google Apps Script (GAS) をバックエンドとし、Vanilla JS + Tailwind CSS で構築されたタイ語学習のための超高速SPA（Single Page Application）。
単なるフラッシュカードアプリにとどまらず、独自のタイ語音節解析エンジンやAIチャットアシスタントを内蔵し、モバイル（特にiOS）での極限のUXを追求したパーソナル学習プラットフォームです。

🌟 Core Features (主要機能)
1. 🧠 忘却曲線ベースの3段階学習エンジン (StudyEngine)
3段階評価システム: カードめくり後、「簡単 (3)」「普通 (2)」「難しい (1)」の3ボタンから忘却度を判定し、次回の出題日を自動計算します。

ヒートマップ可視化: 過去の学習実績（Activity）を frappe.Chart を用いて視覚的にトラッキング可能です。

スマート/クラシックモード: アルゴリズムに基づく出題（🧠）と、単なるランダム（✨）を即座に切り替えられます。

2. ⚡️ 万能検索窓「Omnibox」＆ リアルタイム辞書 (VocabListUI)
超広範インクリメンタルサーチ: タイ語、日本語、発音記号、品詞に加え、「解説文 (explanation)」や「例文 (example)」の中身までリアルタイムに検索し、スニペットとしてハイライト表示（<b class="...">）します。

iOSセキュリティ突破ハック: 画面右下のフローティングボタンに「透明なダミーInput要素」を被せることで、iOS特有のキーボード起動制限を物理的に突破し、1タップで検索窓へフォーカスを移します。

単方向データ同期: ヘッダーの入力欄（Add）とリストの検索窓（Search）で、ペーストやスワイプ開閉時でも値がねじれない堅牢な同期ロジック（VocabListUI.update）を実装しています。

3. ⚙️ タイ語特化の独自解析エンジン (ThaiParser & Validator)
システム内部でタイ語の文字列を音節レベルに分解・解析する強力なエンジンを搭載。

高精度な音節分解: ThaiParser.parseSyllables により、前置母音、核子音、Ro Han (รร)、末子音を左から舐めるように吸い込んで解析します。

声調・生音/死音の自動計算: ThaiValidator が文字のクラス（高・中・低）と母音の長さ、末子音の性質から、タイ語の絶対ルールに従って声調番号（0〜4）を全自動計算します。

自己診断とリカバリー: 裸の子音などを検知した場合、前後の音節と強制吸着を試みる「リカバリー・レイヤー」を持ち、解析エラーの疑いがある場合は画面上に警告（⚠️）を出力します。

カスタム分割対応: どうしても機械的にパースできない例外単語（ทย など）に対しては、ユーザーが手動で分割を指定できる custom_split 機能を備えています。

4. 🤖 AI統合チャット＆動的カード化 (ChatSystem)
アプリ内で直接AI教師（Gemini）と対話できるシート UI を実装。

2つのモード切替: 「🔁 翻訳・カード化（translate）」と「🤖 先生に質問（ask）」のモードを切り替え、目的に応じたプロンプトを送信します。

単語の自動抽出スナイパー: AIの返答テキストから、正規表現（リスト形式やMarkdownテーブル）を用いて「タイ語・発音記号・日本語」のペアを自動抽出し、画面上に登録用リストとして動的に生成します（extractAndRenderVocab）。

重複チェックとジャンプ: 抽出された単語がすでにDBに存在するかチェックし、未登録なら「追加」ボタン、登録済みなら対象のカードへジャンプするボタン（登録済 ↗）へと切り替えます。

画像解析対応: 画像（Base64）をアップロードし、タイ語の解析を依頼する機能（sendImage）を備えています。

5. 🔗 網羅的な相互リンクと逆引きシステム (StudyEngine)
自動コンテキストリンク: 例文（example_th）や解説（explanation）のテキスト内に、既に単語帳に登録されているタイ語が含まれている場合、自動的に青文字のリンク（context-link）に変換します。

タップでツールチップ: リンクを1回タップすると独立したツールチップで意味を表示し、2回タップするとその単語のカードへ直接ジャンプ（jumpToWordFromId）します。

逆引き表示（References）: 今見ている単語を「解説」や「例文」に含んでいる「他の単語カード」を抽出し、カード下部にリンクチップとして表示（renderReferences）します。

6. 🎧 ハイブリッド音声合成 (AudioSystem)
Cloud TTS / Browser TTS のシームレス切替: ヘッダーのボタン（☁️/🤖）で、Google Cloud API経由の高音質音声と、ブラウザ標準の SpeechSynthesis を即座に切り替えます。

自動キャッシュ: Cloud TTSで一度取得した音声データ（Base64）は JS のオブジェクト（audioCache）に保存され、通信量を削減します。

スロー再生対応: 🐢ボタンにより、再生速度（playbackRate / rate）を 0.9倍 から 0.5倍 に切り替える機能（toggleSlowMode）を搭載しています。
